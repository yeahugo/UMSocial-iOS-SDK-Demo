UMSocialCustomAccount Class Reference

Inherits from NSObject
Declared in UMSocialAccountManager.h

Overview

开发者自有账户对象，在app登录或者注册完成之后用用户名来初始化这样的对象，可以指定头像等，然后用UMSocialAccountManager来添加到友盟的账户体系上，使用我们的评论列表和个人中心的页面时就会显示自有账号的用户名和头像
