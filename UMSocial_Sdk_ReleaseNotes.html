iOS Social SDK Release Note

版本 2.1

  • 发布日期： 2013-4-26
  • 主要更新： 去除采集设备标识udid。

版本 2.0

  • 发布日期： 2013-4-11
  • 主要更新： 各个分享微博的接口（包括列表页面、编辑页面、授权页面、个人中心页面）支持新浪微博SSO授权，即如果有安装新浪微博客户端跳转到客户端授权，没有的话使用原来的方式。

版本 1.9

  • 发布日期：2013-4-1
  • 主要更新：
    1. 分享编辑页面支持切换用户账户。
    2. 微信分享支持应用分享。

版本 1.8

  • 发布日期：2013-3-8
  • 主要更新：
    1. 支持分享url的图片、音乐、视频资源到各个微博平台。
    2. 统计分享到微信、Facebook、Twitter、邮件、短信内容，可以在我们的网站报表看到对应的统计数据。

版本 1.7

  • 发布日期：2013-2-4
  • 主要更新：
    1. 提供快速分享接口，简化集成SDK步骤。
    2. 提供分享列表新样式，原来的分享列表提供改为更为灵活的配置方法。
    3. 优化个人中心和评论的登录逻辑。
    4. 用iOS6的系统框架实现Facebook和Twitter分享。

版本 1.6

  • 发布日期：2013-1-5
  • 主要更新：
    1. SDK的库文件分拆为两个，如果没有用评论功能的话，只集成一个库文件，减少了库文件的大小。如果要使用评论功能则需要两个库文件。
    2. 评论列表页面、分享列表页面、个人中心页面均可以配置是否需要登录。
    3. 增加评论列表页的缓存。
    4. 增加社会化操作栏的回调方法，公开操作栏视图文件的源代码，可以自定义按钮样式、响应事件等。

版本 1.5

  • 发布日期：2012-12-4
  • 主要更新：
    1. 全部页面兼容iPad。
    2. 授权页面增加关注官方微博，和关注好友的数据级接口。
    3. 增加配置支持的屏幕方向,各个界面级回调方法，分享编辑页面删除图片。
    4. 修改bug。

版本 1.4

  • 发布日期： 2012-11-9
  • 主要更新：
    1. 各个页面支持横屏。
    2. 分享增加@好友的功能。
    3. 增加获取好友等数据级接口。

版本 1.3

  • 发布日期： 2012-10-18

  • 初始发布：
    我们的组件式社会化服务，为用户提供简约、流畅的社会化分享、评论、登录体验， 并为APP轻松创建一个基于兴趣的互动社区 。

  • 三大看点:

    1. 分享聚合——众多社交平台一站管理。
    2. 轻松满足更多社会化需求——登录、评论、喜欢。
    3. 社会化效果统计报表。
